Question-1: What are the different ways to select an element in the DOM?

There are many ways in which we can select elements in a DOM using selectors. getElementsByTagName(),getElementsByName() , getElementById(), getElementsByClassName() querySelector(), and querySelectorAll() .Use querySelector and querySelectorAll for modern,CSS-style selection,and getElementById,getElementsByClassName.etc for simple use cases.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

In JavaScript ,innerHTML,innerText,innerContent are properties used to access or modify the content of an element. innerHTML use when we want to work with the Html structure inside an element,and it returns or sets all html tags and text. innerText use when we want to care about the text that is visibly rendered on the page,it returns only the text is visiblle to the user. textContent use when we want to work with all the text content,including hidden text.

Question-3: What is event delegation in the DOM?

Event Delegation is basically a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process that propagates events up through the DOM tree, starting with the innermost element. This process is similar to ripples spreading out in a pond.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new element using the document.createElement() method. For adding elements we can use appendChild(), append(), prepend(), insertBefore(), etc.To remove an element use the removeChild() or remove() method.
